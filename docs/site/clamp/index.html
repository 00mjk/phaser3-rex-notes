    Clamp

    Introduction¶

    Force a value within the boundaries by clamping it to the range min, max, built-in method of phaser.

    • Author: Richard Davey

    Usage¶

    var result = Phaser.Math.Clamp(value, min, max);
    • value : The value to be clamped.
