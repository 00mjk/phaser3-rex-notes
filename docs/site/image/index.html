    Image

    Introduction¶

    Display of static images, built-in game object of phaser.

    • Author: Richard Davey

    Usage¶

    Load texture¶

    scene.load.image(key, url);

    See loader

    Add image object¶

    var image = scene.add.image(x, y, key);

    Add image from JSON

    var image = scene.make.image({
    x: 0,
    y: 0,
    key: '',

    // angle: 0,
    // alpha: 1
    // flipX: true,
    // flipY: true
});
