    Phaser¶

    Phaser is a fun, free and fast 2D game framework for making HTML5 games for desktop and mobile web browsers, supporting Canvas and WebGL rendering.

    List of my plugins¶

    1. Canvas: Drawing on canvas.
    2. Clock: A clock to count elapsed time.
    3. Drag: Drag game object.
    4. Drag-cursor: Simulate curosr keys according dragging events.
    5. Gashapon: Pick random item from box.
    6. LZ-string: Compress string using LZ-based compression algorithm.
    7. Page: Display text page by page on text object.
    8. Run-commands: Run commands in array.
    9. TCRP.Player: Run commands on time.
    10. TCRP.Recorder: Store commands with time.
    11. Typing: Typing text on text object.
    12. Webfont-loader: Load web font by google webfont loader in preload stage.
    13. XOR: Encrypt or decrypt string by XOR algorithm.
