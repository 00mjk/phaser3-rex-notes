    Phaser¶

    Phaser is a fun, free and fast 2D game framework for making HTML5 games for desktop and mobile web browsers, supporting Canvas and WebGL rendering.

    List of my plugins¶

    1. BBCodeText: Drawing text with BBCode protocol.
    2. Canvas: Drawing on canvas.
    3. Clock: A clock to count elapsed time.
    4. CSV-to-hash-table: Hash table indexed by (col-key, row-key) from csv string.
    5. Drag: Drag game object.
    6. Drag-cursor: Simulate curosr keys according dragging events.
    7. Fade-out-destroy: Fade out game object then destroy it.
    8. Fade-volume: Fade-in/fade-out volume of sound.
    9. Gashapon: Pick random item from box.
    10. Grid table: Viewer of grid table, to manipulate game object of each visible cell.
    11. LZ-string: Compress string using LZ-based compression algorithm.
    12. Page: Display text page by page on text object.
    13. Run-commands: Run commands in array.
    14. TagText: Displays text with multi-color, font face, or font size with tags.
    15. TCRP.Player: Run commands on time.
    16. TCRP.Recorder: Store commands with time.
    17. Typing: Typing text on text object.
    18. Webfont-loader: Load web font by google webfont loader in preload stage.
    19. XOR: Encrypt or decrypt string by XOR algorithm.
