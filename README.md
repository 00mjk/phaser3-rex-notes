## Demo

### Prepare

1.  install [node.js](https://nodejs.org/en/)
2.  Click `install.bat`
    -   `npm install`

### Run demo

1.  Click `*.bat` in folder `demo`, it will open application on browser

## File structure

-   Folder `plugins` : source code of plugins
-   Folder `demo` : demo code
-   Folder `docs` : notes
