## Header (Required)

- <randomID>
    - `ownerID` - Unique ID of owner
    - `fileID` - Unique file ID of each owner
    - `type` - 'header'
    - `headerDocID` - Document ID of this header
    - `contentDocID` - Document ID of content
    - ...

## Content

Optional

- <randomID>
    - `ownerID` - Unique ID of owner
    - `fileID` - Unique file ID of each owner
    - `type` - 'content'
    - ...
